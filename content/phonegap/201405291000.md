# はじめに
jqplotをレンダリング


phonegapで実行される場合タイミングがずれるので意図した通りに描画されない。

jquery.onを利用して描画のタイミングを設定する。


$(document).on('pageshow','<id>',function(){});

pageshow:
 
id これはjqplotそのものではなく上位の要素に対して実行を行う。
つまりページの中にjqplotのパーツが入っているので
pageshowは親ページを指す。
